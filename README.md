# GamesLab3.

## Описание
Игра в жанре автошахматы с поддержкой сохранения/загрузки и взаимодействием двух игроков.

## Технологии
- Язык: C#
- Фреймворк: .NET
- Тестирование: xUnit
- GUI: WPF / Avalonia UI
